Grid Areas

Al contenedor con display: grid se le aplica la propiedad grid-template-areas, colocando nombres para cada área. Si un item debe ocupar el espacio de varias columnas, el nombre del area debe repetirse por la cantidad de columnas que ocupará el item.

En cada item hijo se debe colocar la propiedad grid-area con el nombre correspondiente colocado en la propiedad grid-template-areas para hacer la relación.
