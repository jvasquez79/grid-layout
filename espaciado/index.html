Espaciado entre items

Al usar márgenes se logra el espaciado pero los items pierden tamaño.

Esto se soluciona con las propiedades grid-column-gap, grid-row-gap y grid-row.

grid-row combina ambas indicando espaciado para filas y luego para columnas.
