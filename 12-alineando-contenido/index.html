Alineando contenido

Se utilizan las propiedades justify-content y align-content para alinear el grid como tal.

  1. start coloca el grid al inicio.
  2. end coloca el grid al final.
  3. center coloca el grid en el centro.
  4. stretch expande para que ocupe todo el espacio disponible.
  5. space-around coloca espacio alrededor de cada item; los espacios entre dos items se suman.
  6. space-between coloca espacio alrededor de los items excepto por el primero y el Ãºltimo.
  7. space-evenly coloca espacio alrededor de cada item, sin sumar los espacios entre dos items subyacentes.
