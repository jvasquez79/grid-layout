Definir tamaño de columnas

Se utilizan las propiedades grid-column, grid-column-start y grid-column-end.

Al usar grid-column, los valores de inicio y fin se separan con una diagonal (/).

Al indicar un valor numérico para el fin, se entiende el número de la línea de grid donde terminará el área, no la cantidad de espacios a colocar.

Para indicar la cantidad de espacios se utiliza span n, donde n es la cantidad de espacios que la columna ocupará.

Si se desconoce la cantidad de columnas, se puede utilizar -1 como valor final para indicar el resto de la fila.
